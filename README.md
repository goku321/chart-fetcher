# IMDB Chart Fetcher

## How To Run

1. `go get github.com/goku321/chart-fetcher` and enter the code directory.

2. `make build`

3. `./chart-fetcher <url> <count>`


    Example:
    ```
    ./chart-fetcher https://www.imdb.com/india/top-rated-indian-movies/ 2
    ```

## How to Run Test

`make test`
